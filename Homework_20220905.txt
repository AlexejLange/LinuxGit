(1) Что делает команда git revert и пример ее использования.
	Отменяет изменение, путём создания нового коммита. Создаёт изменения обратные отменяемому коммиту. 
	Пример импользования: git revert f54j
(2) Какой командой можно изменить последний коммит?
	git commit --amend
(3) Что делает команда git remote rm origin?
	Отменяет привязку к удалённому репозиторию
(4) В каких случаях происходит MERGE CONFLICT?
	Если при слиянии откличается код файла в одной и той же строке, 
	то есть паралельно были сделаны изменения одной и то же строки кода. 
(5) Как посмотреть отличия между ветками?
	git diff branch1 branch2
(6) Какой командой удалить локальную и удаленную ветки?
	git branch -d имя ветки (локально)
	git push origin --delete имя ветки (из удалённого репозитория)
(7) Как сделать файл неотслеживаемым (untracked)?
	git rm --cached имя файла
